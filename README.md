# Go Moto

Кастомная WordPress-тема для сайта сервиса аренды и продажи мототехники. Проект объединяет витринные страницы, каталог на базе WooCommerce, карточки техники и онлайн-бронирование в одном адаптивном интерфейсе.

Фокус проекта - фронтенд-реализация коммерческого сайта: быстрый UI, управляемый контент, интерактивные галереи и аккуратная интеграция с WordPress.

## Основные возможности

- Главная страница с hero-видео, подборками техники, преимуществами, FAQ, галереей и SEO-блоками.
- Каталог WooCommerce с категориями, карточками товаров и сортировкой по году выпуска.
- Детальная карточка мотоцикла с фотогалереей, характеристиками, ценой, похожими предложениями и аудио звука мотора.
- Онлайн-бронирование через встраиваемую форму с ленивой загрузкой внешнего виджета.
- Контактная страница с данными компании, картами, галереей и формой обратной связи.
- Адаптивные слайдеры, mobile-menu, фильтрация категорий, lightbox-галереи и анимированные счетчики.
- JSON-LD schema-разметка для главной страницы, товаров и статей.

## Стек

WordPress, WooCommerce, PHP, Carbon Fields, TypeScript, SCSS, Vite, Swiper, Fancybox, Inputmask, Contact Form 7.

## Особенности реализации

- **Современная сборка для WordPress.** Vite подключает TypeScript и SCSS через manifest в production и поддерживает dev-server с перезагрузкой PHP-шаблонов при разработке.
- **Модульный frontend.** Интерактивные части подключаются динамически только при наличии нужных элементов на странице: слайдеры, галереи, маски форм, счетчики и фильтр категорий.
- **Управляемый контент.** Carbon Fields используется для настроек сайта, главной страницы, товаров, стикеров, контактов, скидок и форм бронирования.
- **Оптимизация загрузки.** Тема отключает лишние стили WordPress/WooCommerce, переносит jQuery в footer, прелоадит ключевые шрифты и главное изображение товара.
- **SEO-ориентированная структура.** Реализована JSON-LD-разметка для организации, товаров и статей, а контентные блоки вынесены в управляемые поля.

## Запуск проекта

```bash
npm install
composer install
npm run dev
```

Для production-сборки:

```bash
npm run build
```

Тема рассчитана на запуск внутри WordPress-окружения с активированными WooCommerce и Carbon Fields.

## Статус проекта

Коммерческий проект, сфокусированный на разработке кастомной WordPress-темы, frontend-архитектуре, адаптивном UI и интеграции коммерческого каталога.
